#  **dotfiles**

[汉语](https://github.com/CuitGGyy/dotfiles)
[English](https://github.com/CuitGGyy/dotfiles/blob/master/README.en_US.md)


## 说明

- 自用 vim / neoVim 配置文件
- 要求 vim 7.4+ 或 neovim 0.7+
- 遵循 XDG 目录规范
- 使用 [vim-plug](https://github.com/junegunn/vim-plug) 管理插件

- 自用 zsh 环境配置
- 自用 bash 环境配置
- 自用 python 环境配置
